### Lesson 3 - automating tasks with Ansible

1. Install Ansible
```
sudo apt-add-repository ppa:ansible/ansible
sudo apt update
sudo apt install ansible

```

2. Install sshpass
```
apt-get install sshpass
```

3. Create files for inventory, vars, configs, etc.
4. Create Ansible playbook
   1. bfdd
   2. bfd config
   3. srv6 config
5. Run playbook
